# 🎮 Найди пару - Игра Памяти

Увлекательная игра для развития памяти с полной системой достижений, челленджей и лидербордов!

## 📋 Структура Проекта

### Основные файлы:
- **index.html** - Главная страница (3 экрана: вход, меню, игра)
- **script.js** - Основная логика игры и аутентификация
- **style.css** - Адаптивный дизайн (690+ строк CSS)
- **integration.js** - Интеграция всех 8 систем

### 8 Интегрированных Систем:

1. **🎖️ achievements.js** - Система достижений
   - 8+ уникальных наград
   - Система очков и прогресса
   - Разблокировка по достижениям

2. **📅 challenges.js** - Челленджи и задачи
   - Ежедневные челленджи
   - Еженедельные задачи
   - Система прогресса

3. **🏆 leaderboards.js** - Лидерборды
   - Глобальные рейтинги (топ-100)
   - Система очков
   - История побед

4. **🎨 cosmetics.js** - Косметика и скины
   - 4+ темы оформления
   - Кастомизация карточек
   - Система покупок

5. **🔀 card-sets.js** - Наборы карточек
   - 🐾 Животные (по умолчанию)
   - 🍕 Еда (заблокирован)
   - 🌲 Природа (заблокирован)
   - ⚽ Спорт (заблокирован)

6. **⏱️ game-modes.js** - Режимы игры
   - 🎮 Классический (неограниченно)
   - ⏱️ На время (2 минуты)
   - 💔 Выживание (3 ошибки)
   - ⚡ Блиц (60 секунд)

7. **⚔️ arena.js** - Онлайн арена
   - Еженедельные турниры
   - Ежемесячные турниры
   - Система призов

8. **👥 social-quests.js** - Социум
   - Система друзей
   - Вызовы между игроками
   - Совместные челленджи

## 🚀 Как Запустить Локально

```bash
# Просто откройте в браузере:
# Запустите локальный сервер (Python):
python3 -m http.server 8000

# Или используйте другой способ (Node.js, PHP и т.д.)
node -e "require('http').createServer((_, r) => r.end(require('fs').readFileSync('index.html'))).listen(8000)"
```

Затем откройте: `http://localhost:8000`

## 💾 Сохранение Данных

Все данные хранятся в `localStorage`:
- `currentUser` - текущий пользователь
- `user_{email}` - данные пользователя
- `global_leaderboard` - глобальный лидерборд
- `user_events_{email}` - события пользователя
